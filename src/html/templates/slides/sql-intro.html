A Brief History

1970 👨‍💼

E.F. Codd publishes paper outlining a model for storing and manipulating data using tables.

'79-'82 🚀

Oracle, Relational Technology, Inc. and IBM put out commercial relational databases

1986 ✅

American National Standards Institute (ANSI) standardizes SQL.

SQL

Programming language designed for managing data held in a relational database management system.

Relational Database

A set of tables used for storing data. Each table has a unique name and may relate to one or more other tables in the database through common values.

A table in a database is a collection of rows and columns.
A row contains data pertaining to a single item or record in a table.
A column contains data representing a specific characteristic of the records in the table.

Relational Database Management System

Software for creating, manipulating, and administering a database.

Popular Open Source RDBMS

mySQL 🛠

  • Popular for its rich features, relative ease of use and community support
  • Has built-in security features, is scalable and speedy
  • Used by a great majority of web-apps

PostgreSQL 🏦

  • Advanced RDBMS
  • Differentiated by its compliance to ANSI standards, its extensibility and reliability
  • Used when data integrity is necessary without excuses or complex, custom procedures must be written
