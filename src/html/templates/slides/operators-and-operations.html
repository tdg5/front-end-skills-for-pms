JavaScript: Operators

op·er·a·tor

/ˈäpəˌrādər/

noun

Reserved syntax consisting of punctuation or alphanumeric characters and carrying out built-in functionality. For example, + indicates the "addition" operator and ! indicates the "not" operator.

JavaScript: Operators

Operator Description Example
= Assign a value to a variable 

name = "John"
+ Add one number to another 

(5 + 4) == 9
- Subtract one number from another 

(5 - 1) == 4
* Multiply one number by another 

(5 * 5) == 25
/ Divide one number by another 

(25 / 5) == 5
% Return remainder of dividing one number by another 

(5 % 4) == 1

Comparison Operators

Operator Description Example
== Test if one value is equal to another 

(5 == 5) == true
!= Test if one value is not equal to another 

(5 != 4) == true
> Test if one number is greater than another 

(5 > 4) == true
>= Test if one number is greater than or equal to another 

(5 >= 4) == true
< Test if one number is less than another 

(5 < 4) == false
<= Test if one number is less than or equal to another 

(5 <= 4) == false
