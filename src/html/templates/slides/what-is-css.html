Cascading Style Sheets

CSS

  • Describes how an HTML document should display visually
  • Cascading describes how rules from one style sheet can override previously defined rules
  • Browsers use the order in which style sheets are included in the document and special selector specificity rules to determine which styles should be displayed

CSS

  • Style sheets are composed of rules . Each rule (or ruleset) is composed of one or more selectors and a block of property declarations.
  • Selectors declare which part of the document a style applies to by matching tags and attributes in the document
  • < a href="https://en.wikipedia.org/wiki/Cascading_Style_Sheets#Declaration_block" target="_blank" title="Wikipedia.org - CSS Declaration Block" > Properties describe which aspects of an element should be styled and how

A sample style sheet


/*
  This selector targets any element that has a class of "call-to-action" to
  change the elements background color to red.
*/
.call-to-action {
  background-color: red; /* Make the background of the button red */
  border: 0;             /* with no border */
  border-radius: 4px;    /* with rounded corners */
  box-shadow: 5px 5px rgba(0, 0, 0, .2);
                         /* ^ with a faint shadow */
  color: white;          /* with white text */
  cursor: pointer;       /* with a pointer-finger mouse cursor */
  font-size: 2em;        /* with text twice the normal size */
  height: 100%;          /* with a height 100% the height of its parent */
  width: 20%;            /* with a width 20% the width of its parent */
}
