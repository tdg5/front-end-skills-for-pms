Cascading Style Sheets

CSS

  • Describes how the HTML document should display visually
  • "Cascading" describes how one stylesheet can override another
    • In most situations, the last stylesheet to define a style wins out over any previously defined styles
  • CSS selectors target which elements to add styles to
  • CSS properties describe what aspects of the element should be styled and how
  • MDN - CSS

A sample stylesheet


/*
  This selector targets any element that has a class of "call-to-action" to
  change the elements background color to red.
*/
.call-to-action {
  background-color: red; /* Make the background of the button red */
  border: 0;             /* with no border */
  border-radius: 4px;    /* with rounded corners */
  box-shadow: 5px 5px rgba(0, 0, 0, .2);
                         /* ^ with a faint shadow */
  color: white;          /* with white text */
  cursor: pointer;       /* with a pointer-finger mouse cursor */
  font-size: 2em;        /* with text twice the normal size */
  height: 100%;          /* with a height 100% the height of its parent */
  width: 100%;           /* with a width 100% the width of its parent */
}
