Github

Web-based hosting service for version control using git
Developers are constantly changing code and git helps manage revisions
Provides a graphical interface with project management features

Repository

Location where all the files for a particular project are stored

git clone REPO_NAME

Branch

A copy of the repo where new revisions can take place

git checkout -b BRANCH_NAME

Revisions

To persist changes, we commit them to the local branch

git add FILE_NAME(S)
git commit -m "MESSAGE"

Pull Request

Once the branch is ready, it gets pushed remotely and a pull request is opened. A PR is an opportunity for others to review the code
git push -u origin BRANCH_NAME

Github Projects

A lean alternative to JIRA or Trello to track work
