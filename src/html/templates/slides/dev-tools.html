Elements Panel

Lets you inspect the DOM tree and edit DOM nodes on the fly

There are several ways to inspect an element:

  1. Right-click any element on the page and select Inspect Element.
  2. Press Ctrl + Shift + C (or Cmd + Shift + C on mac), then click on an element.
