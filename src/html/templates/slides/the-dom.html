The Document Object Model

  • Bridges the gap between JavaScript and the world of HTML and CSS
  • API for interacting with the browser's representation of an HTML document
  • Provides full control over the HTML document, enabling the dynamic web

The Document Object Model

Allows JavaScript code to:

  • Create, interact with, and remove elements
  • Change CSS styles
  • Generate and listen for new and existing events
