The Document Object Model

  • Allows JS to interact with the browser's representation of a document
  • Provides full, dynamic control over the HTML document
  • Bridges the gap between HTML, CSS, and JavaScript

The Document Object Model

Allows JavaScript code to:

  • Create, interact with, and remove elements
  • Change CSS styles
  • Generate and listen for new and existing events
