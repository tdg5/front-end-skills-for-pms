HTML

Hyper Text Markup Language

HTML

  • Standard markup language used to create web pages
  • Describes the structure of a website, including cues for presentation
  • Web browsers transform HTML documents into visible (or audible!) web pages

HTML

  • Written in the form of HTML elements which consist of special tags enclosed in angle brackets

  • Elements can include data other than their enclosed content in the form of attributes and values
  • Browsers don't display HTML tags, but use them to interpret the content of the document
