HTML

Hyper Text Markup Language

HTML

  • Standard markup language used to create web pages
  • Describes the structure of a website semantically along with cues for presentation
  • Web browsers transform HTML documents into visible or audible web pages

More information:

  • Wikipedia.org - HTML

HTML

  • Written in the form of HTML elements which consist of tags enclosed in angle brackets (like <article>)
  • Browsers don't display HTML tags, but use them to interpret the content of the document

A simple HTML document


<html>
  <!--
    The <html> element holds the two main components of an HTML document,
    the <head> element and the <body> element.
  -->
  <head>
    <!--
      The <head> element usually includes metadata about the document such
      as title, styling, and other resources required by the document.
    -->
    <title>Hello, World!</title>
  </head>
  <body>
    <!-- The <body> element holds the content of the document. -->
    <div>Hello, World!</div>
  </body>
</html>
